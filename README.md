# 고려대학생 교양 강의 추천 웹페이지
## 각자 취향에 맞는 교양 강의를 추천해드립니다.

**무슨 강의를 들어야할지 몰라서 방황하는 대학생, 내 관심 분야의 강의 목록을 찾고 싶은 대학생들**

- 학생 정보 입력 (학번, 비밀번호)
- 강의 관심 키워드 설정 (오전수업, 저녁수업, 명강, 온라인강의 등)
- 관심 분야 설정 (인문, 자연과학, 외국어, 컴퓨터 등)
- 이후 즉시 매칭되는 강의 목록 제공!
